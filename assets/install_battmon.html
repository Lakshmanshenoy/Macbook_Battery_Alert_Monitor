BattMon Installer Guide

Install BattMon in a few seconds.

BattMon is a lightweight macOS battery monitor. Drag the app into Applications, then open it from there.

Install
  1. Drag BattMon.app to Applications.
  2. Open Applications/BattMon.app.
  3. If macOS warns you, choose Open Anyway in Privacy & Security.
Why the warning appears

BattMon is open-source and distributed outside the Mac App Store. On unsigned early-stage releases, macOS shows a Gatekeeper warning before first launch.

Safe to verify

Release downloads include checksums.txt. You can verify the DMG before opening it.

Troubleshooting

If the app is still blocked, right-click BattMon.app and choose Open once, or use xattr -dr com.apple.quarantine /Applications/BattMon.app for local testing only.
